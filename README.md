# G-Port+

学習院大学生向けの非公式ポータルサイトです。

## 主な機能
- 公式リンク集（検索・カテゴリ絞り込み・ピン留め）
- 時間割の保存
- 今日の授業 / 次の授業の自動表示
- 課題の締切管理
- 今日のメモの自動保存
- ダーク / ライト切り替え

## 公開方法
1. GitHub の Settings > Pages を開く
2. Source を Deploy from a branch にする
3. main / (root) を選ぶ
4. 数分後に公開URLへアクセスする

## 注意
このサイトは非公式です。リンク先や大学側の仕様変更があれば適宜更新してください。
